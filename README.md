# ATS Pro

ATS Pro is a modern Applicant Tracking System (ATS) web application designed to streamline the recruitment process. The platform helps manage job applications, resumes, and candidate tracking efficiently.

## Features
- User Authentication
- Resume Upload & Management
- Candidate Tracking
- Job Listings
- Dashboard Interface
- Responsive UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js / Express.js
- Database: MongoDB
- Styling: CSS / Tailwind

## Installation

```bash
git clone https://github.com/PatraShruti/ats-pro.git
cd ats-pro
npm install
npm start
